# 串口测试硬件清单

## 必需硬件

1. **MSPM0G3507 核心板** × 1
   - 确认型号：MSPM0G3507（48-pin封装）
   - 确认有板载CH340串口芯片

2. **USB数据线** × 1
   - ⚠️ 必须是**数据线**，不能是只充电的线
   - 建议：手机原装数据线

3. **电脑** × 1
   - Windows 10/11
   - 已安装 VSCode + EIDE

## 可选硬件

4. **LED灯** × 1（用于观察程序运行）
5. **杜邦线** × 若干（连接外设）

---

## 硬件连接示意图

```
┌─────────────────────────────────┐
│   MSPM0G3507 核心板              │
│                                  │
│  ┌──────────────┐                │
│  │  MSPM0G3507  │                │
│  │   芯片       │                │
│  │              │                │
│  │  PA10 ─────────→ CH340 (TX)  │
│  │  PA11 ←─────────  CH340 (RX)  │
│  └──────────────┘                │
│                                  │
│          CH340                   │
│       (USB转串口芯片)            │
│            ↓                     │
│        USB接口 ──────────────────┼──→ 连接电脑
│                                  │
└─────────────────────────────────┘
```

说明：
- PA10 (UART0_TX) 已经连接到CH340的RX
- PA11 (UART0_RX) 已经连接到CH340的TX
- 核心板设计已完成连接，**无需额外接线**
