# MSPM0G3507 循迹小车参数调试指南

## 📋 调试步骤

### 第一阶段：基础循迹调试

#### 1. 初始参数设置
```c
BASE_SPEED_STRAIGHT = 400   // 从低速开始
KP_STRAIGHT = 30
KD_STRAIGHT = 15
```

#### 2. P参数调试（比例控制）
- **现象**：小车左右摇摆
- **调整**：
  - 摇摆幅度大 → 减小 Kp（如30→25）
  - 响应太慢、压线 → 增大 Kp（如30→35）
  
**目标**：小车能跟线，但可能会有轻微摇摆

#### 3. D参数调试（微分控制）
- **现象**：小车摇摆后不能快速稳定
- **调整**：
  - 振荡频繁 → 增大 Kd（如15→20）
  - 过度阻尼、反应迟钝 → 减小 Kd（如15→10）
  
**目标**：小车平稳跟线，摇摆快速收敛

#### 4. 速度提升
固定好PID参数后，逐步提升速度：
```
400 → 500 → 600 → 700 → 800
```
每次提升重新微调PID，弯道可能需要更大的Kd。

---

### 第二阶段：弯道优化

#### 1. 弯道检测阈值
```c
CORNER_DETECT_THRESHOLD = 150  // 陀螺仪角速度阈值
```

**调试方法**：
- 让小车跑一圈，串口输出陀螺仪数据
- 观察直道和弯道的角速度差异
- 阈值设在两者中间

#### 2. 弯道减速
```c
BASE_SPEED_CORNER = BASE_SPEED_STRAIGHT * 0.6~0.8
```

- 速度过快冲出去 → 降低弯道速度
- 速度太慢影响时间 → 适当提高

#### 3. 弯道PID参数
弯道通常需要：
- **更大的Kp**（快速纠偏）
- **更大的Kd**（抑制过冲）

```c
KP_CORNER = KP_STRAIGHT * 1.2~1.5
KD_CORNER = KD_STRAIGHT * 1.3~1.6
```

---

### 第三阶段：陀螺仪融合

#### 1. 陀螺仪辅助增益
```c
GYRO_ASSIST_GAIN = 10
```

**作用**：抑制转向过冲

**调试**：
- 弯道甩尾严重 → 增大增益（10→15）
- 转向响应变慢 → 减小增益（10→5）
- 如果效果不明显，可设为0禁用

#### 2. 陀螺仪校准
每次启动必须静止校准：
```c
ICM42688_CalibrateGyro(1000, 2);  // 1000次采样，±2°/s容差
```

---

### 第四阶段：速度平滑

#### 1. 加速度限制
```c
MAX_ACCELERATION = 50  // 每个控制周期最大变化
```

**效果**：防止速度突变导致打滑

**调试**：
- 加速太慢 → 增大（50→80）
- 仍有打滑 → 减小（50→30）

#### 2. 误差滤波
```c
ERROR_FILTER_ALPHA = 0.7  // 滤波系数
```

- 值越大，响应越快，抖动越多
- 值越小，越平滑，但有延迟

## 🎯 针对发挥部分(3)的特殊优化

### 任务：边跑边画圆，要求极高平稳性

#### 1. 降低速度，提升稳定性
```c
BASE_SPEED_STRAIGHT = 400  // 降低到400-450
BASE_SPEED_CORNER = 350
```

#### 2. 增加控制频率
```c
CONTROL_PERIOD_MS = 2  // 从5ms降到2ms
```

#### 3. 编码器闭环（关键！）
```c
// 四个电机独立速度闭环
for (int i = 0; i < 4; i++) {
    actualSpeed[i] = encoder[i].read();
    motorPWM[i] = SpeedPID(targetSpeed[i], actualSpeed[i]);
}
```

#### 4. 位置估计（用于云台同步）
```c
// 根据编码器和陀螺仪融合估计位置
posX += (leftSpeed + rightSpeed) / 2 * cos(heading) * dt;
posY += (leftSpeed + rightSpeed) / 2 * sin(heading) * dt;
heading += gyroZ * dt;
```

## 🐛 常见问题排查

### 1. 小车偏离轨迹
- **检查**：循迹传感器是否正常
- **检查**：四个轮子速度是否一致（用编码器验证）
- **检查**：地面是否平整

### 2. 转弯冲出去
- **原因**：速度太快或PID响应不够
- **解决**：降低弯道速度，增大Kp和Kd

### 3. 直道摇摆
- **原因**：Kp过大或Kd不足
- **解决**：减小Kp或增大Kd

### 4. 陀螺仪漂移
- **原因**：未校准或温度漂移
- **解决**：
  - 每次启动重新校准
  - 使用互补滤波或卡尔曼滤波

### 5. 速度不稳定
- **原因**：电池电压下降或电机负载不均
- **解决**：
  - 使用编码器闭环控制
  - 电池电压补偿

## 🧪 测试流程

### 阶段1：低速测试（速度400）
1. 能完整跑1圈不掉线 ✓
2. 能稳定跑5圈 ✓
3. 直道和弯道平稳过渡 ✓

### 阶段2：提速测试（速度600）
1. 1圈<20s ✓
2. 2圈<40s ✓
3. 转弯不冲出 ✓

### 阶段3：瞄准联调
1. 边跑边瞄准靶心 ✓
2. 画圆同步误差<1/4圈 ✓
